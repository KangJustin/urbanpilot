# UrbanPilot

A multi-agent urban planning copilot built at **Berkeley AI Hackathon 2026** (June 20–21, 2026).

UrbanPilot analyzes neighborhoods, resolves tradeoffs across climate, accessibility, housing, and urban design, and generates interactive future-city scenarios for 2040 and 2075.

> Frontend inherited from GreenPlanner (MIT Hacks) — Create React App, Leaflet, React Leaflet, Tailwind.

## Getting Started

```bash
npm install
npm start       # http://localhost:3000
npm run build
```
